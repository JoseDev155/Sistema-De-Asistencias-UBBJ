# Descripción de la Documentación
En está sección se proporciona una descripción general de la documentación del proyecto "Sistema de Asistencias UBBJ". La documentación está organizada para facilitar la comprensión y el uso del sistema, cubriendo aspectos clave como la arquitectura, las tecnologías utilizadas, y las funcionalidades principales.

## Estructura de la Documentación
La documentación se divide en varias secciones, cada una enfocada en un aspecto específico del sistema.

**NOTAS:**
>Los archivos `*-ME.md` son mis notas personales para cada sección, no entregar como parte de la documentación oficial :p
>
>Los archivos `* - Template.*` son plantillas vacías para cada sección, por si los quieren usar como base para escribir la documentación oficial en caso mi redacción no sea acertada
